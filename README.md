<h1>djangoblog-kk</h1>

<h1>はじめに</h1>
ブログ管理アプリケーションです。<br>
勉強の為、今後学習したいPythonを使いアプリを作成。

- アプリ名:KawakamiBlog
- 言語:HTML, Python 3.9.12
- フレームワーク:Bootstrap, Django
- DB:SQlite3

<h1>追加した機能</h1>

- CRUD
- ユーザー登録
- ログイン・ログアウト
- ログイン時のみ投稿可能
- 自分の投稿のみ編集、削除可能
- サムネイル画像の追加


<h1>ライブラリインストール</h1>

```
git install django
git install pillow
```
